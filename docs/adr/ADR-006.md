# ADR-006: Support Klingon and Two Dialects of Elvish

## Status

Accepted

## Context

The Enterprise FizzBuzz Globalization Directive mandates support for all
spacefaring civilizations. Additionally, the platform's commitment to
linguistic equity requires representation of both the Grey-elven and
High-elven communities, whose distinct linguistic traditions must not be
collapsed into a single "Elvish" locale.

The existing localization subsystem (`enterprise_fizzbuzz/infrastructure/i18n.py`)
already supported four terrestrial locales: English (`en`), German (`de`),
French (`fr`), and Japanese (`ja`). The `.fizztranslation` file format
(see ADR-002) was designed with extensibility in mind.

The question was not whether to support constructed and fictional languages,
but which ones, and with what level of linguistic rigor.

## Decision

We added three locale files using ISO 639-3 codes:

### `locales/tlh.fizztranslation` â€” Klingon (tlhIngan Hol)

- **Locale code**: `tlh`
- **Fallback**: `en`
- **Plural rule**: `0` (always "other" â€” Klingon, like Japanese, does not
  grammatically distinguish singular and plural forms)
- **Labels**: Fizz = `ghum`, Buzz = `wab`, FizzBuzz = `ghumwab`
- **Banner subtitle**: `Y O '   C H E N M O H W I '`
- The file header reads: `Heghlu'meH QaQ jajvam!` ("Today is a good day to
  FizzBuzz!")

### `locales/sjn.fizztranslation` â€” Sindarin (Grey-elven)

- **Locale code**: `sjn`
- **Fallback**: `en`
- **Plural rule**: `n != 1`
- **Labels**: Fizz = `Hith`, Buzz = `Glamor`, FizzBuzz = `HithGlamor`
- **Plurals**: Buzz pluralizes via i-affection: `Glamor` (sg.) / `Glemyr` (pl.)
- **Linguistic attestation**: `hith` (KHITH, Ety/364), `glamor` (GLAM, Ety),
  `glemyr` (i-affection plural)
- **Sources**: Eldamo, Parf Edhellen, Tolkien Gateway, RealElvish Academy
- **Banner subtitle**: `E C H O R   E N   U D U N`

### `locales/qya.fizztranslation` â€” Quenya (High-elven)

- **Locale code**: `qya`
- **Fallback**: `en`
- **Plural rule**: `n != 1`
- **Labels**: Fizz = `WingĂ«`, Buzz = `LĂ¡ma`, FizzBuzz = `WingeLĂ¡ma`
- **Plurals**: Fizz pluralizes `WingĂ«` / `Wingi`; Buzz pluralizes `LĂ¡ma` /
  `LĂ¡mar`
- **Linguistic attestation**: `wingĂ«` (GWIG/WIGIL, Ety/PE17), `lĂ¡ma` (LAM,
  "ringing sound")
- **Sources**: Eldamo, Parf Edhellen, Tolkien Gateway, RealElvish Academy
- **Banner subtitle**: `A R A N I Ă‹   T E N G W E S T A`

All three locales implement the full section set required by the
`.fizztranslation` specification: `[labels]`, `[plurals]`, `[messages]`,
`[summary]`, `[banner]`, and `[status]`. All three fall back to English when
a key is not found, via the `LocaleResolver` fallback chain.

The `PluralizationEngine` in `i18n.py` registers built-in rules for all three
locales:

```python
_BUILTIN_RULES = {
    "tlh": "n != 1",
    "sjn": "n != 1",
    "qya": "n != 1",
}
```

Note: Klingon's built-in rule is `n != 1`, but the `tlh.fizztranslation` file
overrides `@plural_rule = 0`, which means the engine always returns `other`.
The built-in rule exists for consistency; the file-level directive takes
precedence.

The Elvish locales were subjected to a linguistic validation process.
Attestations are cited in each file's header comments, referencing canonical
sources: the Etymologies (Ety), Parma Eldalamberon (PE17), and community
lexicographic databases. A linguistic validation report was produced to
document the attestation methodology and any corrections made.

During validation, it was discovered that an earlier draft used a Sindarin word
for "Strategy" that was, in fact, an epithet of Sauron. Corrections were issued.

## Consequences

**Positive:**

- The platform now supports 7 locales across 4 language families (Germanic,
  Romance, Japonic, and Constructed), covering 3 planets and 2 fictional
  continents.
- Klingon and both Elvish dialects are linguistically attested. Word choices
  are sourced from published Tolkien manuscripts and community lexicographic
  databases. This is not lorem ipsum in tengwar.
- Sindarin demonstrates i-affection pluralization (`Glamor` / `Glemyr`);
  Quenya demonstrates regular nominal pluralization (`LĂ¡ma` / `LĂ¡mar`).

**Negative:**

- The number of users who will evaluate FizzBuzz in Klingon, Sindarin, or
  Quenya is estimated at zero.
- A linguistic validation report was necessary. This is a consequence that
  belongs to no other FizzBuzz implementation in recorded history.
- The Sauron incident required a corrective action item involving First Age
  Sindarin etymology.

**Neutral:**

- The total locale count is now 7, which is a prime number. This is coincidental
  but noted.
- All three constructed-language locales fall back to English. The fallback
  chain works identically for Klingon as it does for French, because the
  `LocaleResolver` does not discriminate on the basis of a language's
  ontological status.
